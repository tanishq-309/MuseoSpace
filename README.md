Welcome to MuseoSpace, an immersive web-based virtual art exhibition that transforms a traditional online gallery into a cinematic storytelling experience. This project simulates the journey of a visitor through a curated exhibition, using scroll-driven transitions, interactive artworks, and user engagement, all built with pure HTML, CSS, and JavaScript.

This project was built for a hackathon with the constraint of using **no frameworks or databases**.

---

## âœ¨ Features

* **Cinematic Lobby:** An immersive landing page with ambient visuals and a "Start the Tour" call to action.
* **Collection-Based Gallery:** A multi-step gallery where users navigate through different collections (Renaissance, Abstract, Modern) one by one.
* **Dynamic Artwork Modal:** Click on any artwork to open a detailed modal view, dynamically populated from a JavaScript data object.
* **Polished Effects:** Features subtle hover-to-zoom effects on modal images and elegant page transitions.
* **Feedback System:** A complete, client-side validated form for visitors to leave feedback.
* **Functional "Share" Button:** Uses the modern Clipboard API to copy the exhibition link.
* **Fully Responsive:** Designed mobile-first, ensuring a seamless experience on all devices.

---

## ðŸ’» Tech Stack

This project is built using **vanilla front-end technologies** only, per the project constraints.

* **HTML5:** Semantic HTML for structure and accessibility.
* **CSS3:** Modern CSS (Flexbox, Grid, Keyframe Animations, CSS Variables) for styling and layout.
* **JavaScript (ES6+):** Pure vanilla JavaScript for all interactivity, including:
    * Dynamic DOM manipulation (populating the gallery and modals).
    * Event handling for navigation and interactivity.
    * Client-side form validation.
    * Navigator.clipboard API for the "Share" button.

## ðŸš€ Getting Started

This is a fully static project. No build tools or servers are required to run it.

### Prerequisites

You only need a modern web browser.

### Installation & Setup

1.  **Clone the repository:**
    ```sh
    git clone [https://github.com/your-username/MuseoSpace.git](https://github.com/your-username/MuseoSpace.git)
    ```

2.  **Navigate to the project folder:**
    ```sh
    cd MuseoSpace
    ```

3.  **Run the project:**
    * Simply **open the `index.html` file** in your web browser.
    * *Alternatively*, for the best experience (and to avoid any potential file-path issues), you can run it with a simple local server. If you have VS Code, the **"Live Server"** extension is a great option.
